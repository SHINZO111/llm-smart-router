📦 インストール手順
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  前提条件の確認
─────────────────────────────────────────────────────
    □ Python 3.9 以上がインストールされていること
    □ Windows 10/11 または macOS 12+
    □ (推奨) LM Studioがインストールされていること

    Pythonバージョン確認:
    > python --version

2️⃣  リポジトリのクローン/展開
─────────────────────────────────────────────────────
    既に展開済みの場合はスキップしてください

3️⃣  自動インストール（推奨）
─────────────────────────────────────────────────────
    Windowsの場合:
    > run_gui.bat

    初回起動時に自動的にインストールされます

4️⃣  手動インストール
─────────────────────────────────────────────────────
    自動インストールが失敗した場合:

    # 仮想環境作成
    > python -m venv venv

    # 仮想環境有効化
    > venv\Scripts\activate  (Windows)
    > source venv/bin/activate  (macOS/Linux)

    # 依存関係インストール
    > pip install -r requirements-gui.txt

5️⃣  起動
─────────────────────────────────────────────────────
    # GUI起動
    > python src\gui\main_window.py

🔧 トラブルシューティング
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 「python が見つかりません」
A: Pythonを https://python.org からインストールしてください

Q: 「pip が見つかりません」
A: Pythonインストール時に「Add Python to PATH」にチェックしてください

Q: 「ModuleNotFoundError」
A: 依存関係を再インストール: pip install -r requirements-gui.txt --force-reinstall

Q: 「DLL load failed」
A: Microsoft Visual C++ Redistributable をインストールしてください

🔐 APIキー設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. https://console.anthropic.com/ でAPIキーを取得
2. GUI初回起動時に自動的に設定ダイアログが表示されます
3. またはメニュー「設定」→「APIキー設定」から入力

APIキーは自動的に暗号化され、OS標準のキーストアに保存されます

📊 使用方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 入力欄に質問やタスクを記入
2. 必要に応じてプリセットを選択（CM業務、推し活など）
3. 「実行」ボタンまたは Ctrl+Enter で実行
4. 結果が出力欄に表示されます

詳細は README_GUI.md を参照
